# HTML

## 文件后缀为.html 或者.htm ,不区分大小写
```
<html>
    <head></head>
    <body>
        This is my first HTML. 
        <h1>This is my first HTML.</h1> 
        <h2>This is my first HTML.</h2> 
        <h3>This is my first HTML.</h3> 
        <h4>This is my first HTML.</h4> 
        <h5>This is my first HTML.</h5> 
        <h6>This is my first HTML.</h6>
        <hr /> <!-- 分割线 -->
        <p> &nbsp; This is my first HTML.</p>
        <p> &emsp; This is my first HTML.</p>
        <p>This is my first HTML.</p>
        <b>This is my first HTML.</b>
        <i>This is my first HTML.</i>
        <u>This is my first HTML.</u>
        <b><i><u>This is my first HTML.</u></i></b>
        <pre>This is my 
        first HTML. 
        </pre>
        </br>
        <del>This is my first HTML.</del>
        <font color="coral" size="7" face="宋体">This is my first HTML.</font>
        <!-- 无序列表 -->
        <ul>
            <li>1</li>
            <li>1</li>
            <li>1</li>
            <li>1</li>
            <li>1</li>
        </ul>
        <!-- 有序列表 -->
        <ol>
            <li>1</li>
            <li>1</li>
            <li>1</li>
            <li>1</li>
            <li>1</li>
        </ol>
        
    </body>
</html>
```
